# 02 — Convert Text to Sentence Case

> 🇧🇷 [Português](#português) | 🇺🇸 [English](#english)

---

## English

### About

A Python script that reads a text file, converts its content to sentence case (capitalizing the first letter of each sentence), and saves the result in a new file.
**Example:**

```
once upon a time, in a faraway kingdom...  →  Once upon a time, in a faraway kingdom...
she had skin as white as snow...           →  She had skin as white as snow...
```

### How to Run

1. Place `snowwhite.txt` in the same directory as `main.py`
2. Run the script:

```bash
python main.py
```

3. A new file `snowwhite_corrected.txt` will be created in the same directory with the corrected text.

### Libraries Used

## No external or standard library imports required.

## Português

### Sobre

Um script Python que lê um arquivo de texto, converte seu conteúdo para sentence case (primeira letra de cada frase em maiúscula) e salva o resultado em um novo arquivo.
**Exemplo:**

```
once upon a time, in a faraway kingdom...  →  Once upon a time, in a faraway kingdom...
she had skin as white as snow...           →  She had skin as white as snow...
```

### Como Executar

1. Coloque o arquivo `snowwhite.txt` no mesmo diretório que `main.py`
2. Execute o script:

```bash
python main.py
```

3. Um novo arquivo `snowwhite_corrected.txt` será criado no mesmo diretório com o texto corrigido.

### Bibliotecas Utilizadas

Nenhuma biblioteca externa ou da biblioteca padrão é necessária.
