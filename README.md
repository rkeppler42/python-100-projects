# 🐍 Python 100 Projects — Solo Builds

## About

50 independent Python projects built as part of the Udemy course [**Learn Python by Doing: 100 Projects**](https://www.udemy.com/course/learn-python-by-doing-with-100-projects/) by Ardit Sulce.

The course methodology works in pairs: the instructor builds an application in a guided video, then the student builds a **similar but independent project** from scratch — relying only on notes and memory, no copying. This repository contains exclusively those independent builds.

## What this covers

The projects span a wide range of Python applications, progressively increasing in complexity. Topics include (but are not limited to):

- Core Python: variables, data types, functions, loops, conditionals
- File handling and data processing
- Web development basics
- And more across 50 sections

## Structure

```
.
├── 01-project-name/
│   ├── main.py
│   └── README.md
├── 02-project-name/
│   └── ...
└── README.md
```

Each project folder contains its own `README.md` with a brief description and instructions to run it.

## Sobre

50 projetos Python construídos de forma independente como parte do curso Udemy [**Learn Python by Doing: 100 Projects**](https://www.udemy.com/course/learn-python-by-doing-with-100-projects/) de Ardit Sulce.

A metodologia do curso funciona em pares: o instrutor constrói uma aplicação em vídeo, e depois o aluno constrói um **projeto similar, porém independente**, do zero — usando apenas as anotações feitas durante o vídeo, sem copiar código. Este repositório contém exclusivamente esses projetos autorais.

## O que é abordado

Os projetos cobrem uma ampla variedade de aplicações Python, com complexidade crescente ao longo das seções. Os tópicos incluem (mas não se limitam a):

- Python básico: variáveis, tipos de dados, funções, loops, condicionais
- Manipulação de arquivos e processamento de dados
- Fundamentos de desenvolvimento web
- E muito mais ao longo de 50 seções
